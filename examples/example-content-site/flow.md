# Example Flow — Content Site

## Idea
Build a content or documentation site that needs structure before implementation.

## User
Readers who need clarity, navigation, and trust.

## MVP
- site purpose
- page structure
- navigation model
- first content set
- release checklist

## Example milestone split
1. clarify purpose and audience
2. define information architecture
3. draft main pages
4. implement site
5. polish and handoff
