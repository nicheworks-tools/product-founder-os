# Example Flow — Static Tool

## Idea
Build a small browser-only tool that helps users clean pasted text.

## User
People who want a quick utility with no sign-in.

## MVP
- paste input
- transform text
- copy output
- explain what the tool does
- handle empty state and error-like state cleanly
