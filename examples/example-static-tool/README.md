# Example Static Tool

This is a worked example showing how product-founder-os can be used for a small browser-only utility.

## Example product
A text cleanup tool that reformats pasted text into cleaner output.

## Why this example exists
It demonstrates a complete small-product flow with:
- a seed
- a brief
- a functional spec
- a roadmap
- a current sprint
- a done definition
