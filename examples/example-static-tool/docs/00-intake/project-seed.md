# Project Seed

## Current idea
Build a browser-only text cleanup tool that helps users paste messy text and produce cleaner output.

## Who it is for
People who want a quick utility without sign-in or setup.

## Problem
Small text cleanup tasks are common, but opening a full editor is too heavy.

## MVP target
- paste text
- clean text
- show result
- copy result

## Constraints
- browser-only
- fast
- simple single-page UI
