# Product Brief

## Product
Lightweight SaaS example

## One-line description
A narrow SaaS-shaped product used to demonstrate roadmap continuity, scoped delivery, and change handling.

## Primary user
A repeat user with an ongoing workflow problem.

## Primary value
A durable product workflow with clearer milestone planning than a one-page utility.

## MVP
- one clear recurring workflow
- one primary success path
- explicit scope boundaries
- milestone-aware planning
