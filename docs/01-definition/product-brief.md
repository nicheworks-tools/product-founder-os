# Product Brief

## Product
product-founder-os

## One-line description
A repo operating system for turning product intent into structured product work with GPT and/or Codex.

## Why this should exist
Many AI-assisted repos are weak because they lack a durable operating structure.
This repo exists to give product work a repeatable shape so sessions do not reset from zero.

## Primary user
A builder who wants AI to help develop a real product over multiple sessions.

## Primary value
Clear product definition, roadmap continuity, change handling, and more honest completion tracking.

## MVP
A public repo with:
- clear README / SKILL / AGENTS
- coherent docs structure
- usable definition/planning docs
- minimal helper scripts
- example flows
