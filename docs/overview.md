# Overview

`product-founder-os` is a repo operating system for product development with GPT and/or Codex.

It gives the project a durable structure so that work does not have to restart from scratch every time the conversation resumes.

## Core idea

The assistant should not behave like a one-shot generator.

Instead, it should:

- understand the product
- define the product
- plan the work
- execute against a current plan
- absorb changes cleanly
- move toward a real done state

## Practical value

This repo helps reduce:

- vague starts
- drifting scope
- inconsistent docs
- random implementation order
- fake completion signals
